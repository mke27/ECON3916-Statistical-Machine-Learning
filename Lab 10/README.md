Spurious Correlation and Multicollinearity in Macroeconomic Time Series

This project examines common pitfalls in macroeconomic data analysis, specifically spurious correlation and multicollinearity in non-stationary time series. Using macroeconomic indicators retrieved from the FRED API, I analyzed relationships in raw level data using Python, pandas, and seaborn, demonstrating how trending variables can produce misleadingly strong correlations. I then applied Variance Inflation Factor (VIF) diagnostics using statsmodels to quantify multicollinearity and identify redundant predictors. To address these issues, the non-stationary series were transformed into year-over-year (YoY) growth rates, reducing trend-driven correlation. Finally, I constructed Directed Acyclic Graphs (DAGs) to represent plausible structural relationships among the variables, highlighting how causal reasoning can improve interpretation beyond simple correlation analysis. This workflow illustrates best practices for diagnosing and mitigating statistical artifacts in macroeconomic time-series modeling.
